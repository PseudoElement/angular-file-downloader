How to use this API?


This api offers you to generate text files(including sql-files) with specific table structure.

You can manually add columns and configurate its with providen properties.


List of properties

  • Document type - type of file you want to create.
  • Document name - file name.
  • Add table creation query* - add query for creation new table wtih providen columns.
  • Column name - name of column in table.
  • Max - for columns with type NUMBER - max available number, with type STRING - max string length.
  • Min - for columns with type NUMBER - min available number, with type STRING - min string length.
  • % of rows with nullable value - percent of rows in column with value 'null' or empty value.
  • Foreign key table* - if you need to reference column's value to another table, use this property.
  • Foreign key column* - used together with 'Foreign key table' property and specifies name of column you want to reference created column.
  • Set as primary key* - add PRIMARY KEY flag for created column.

Hints

  • * - only sql table property.
